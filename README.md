# sync-uni-wxcomponents
> Sync uniapp node_modules for mp-weixin components.

## installation
```shell
# public
npm i -g @jswork/sync-uni-wxcomponents
```

## configrc
```json
{
  "app": {
    "page.json": "src/page.json",
    "wxcomponents": "src/wxcomponents"
  },
  "packages": [
    "@alo7/mp-components/miniprogram_dist"
  ]
}
```
